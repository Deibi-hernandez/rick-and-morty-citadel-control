# 🪐 Citadel Variant Control Mainframe

Bienvenido al Mainframe de la Ciudadela. Este sistema de cuadrante permite a los Científicos de la Ciudadela monitorear las distintas variantes multiversales en tiempo real

## 🛠️ Stack Tecnológico
- **Core:** React JS (Componentes funcionales y Hooks avanzados)
- **Build Tool:** Vite
- **Estilos:** Bootstrap & Bootstrap Icons
- **Data Source:** Rick and Morty API
- **Persistencia:** LocalStorage del Navegador
